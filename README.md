## Описание

**CSV Report Generator** — настольное приложение для загрузки, объединения и анализа данных из нескольких CSV-файлов.

Программа позволяет:

- загружать один или несколько CSV-файлов;
- объединять записи в общий набор данных;
- группировать данные по выбранному полю;
- рассчитывать статистические характеристики для числовых данных;
- сортировать сформированные результаты;
- формировать горизонтальные и процентные ASCII-диаграммы;
- экспортировать результаты в форматы CSV и TXT.

## Демонстрационные данные

В каталоге `data/` находятся демонстрационные CSV-файлы, которые можно использовать для проверки работы приложения.

Примеры сценариев использования:

- группировка записей по категории или другому текстовому полю;
- расчёт статистических характеристик по выбранному числовому полю;
- сортировка сформированных групп;
- построение горизонтальных и процентных ASCII-диаграмм;
- экспорт результатов анализа в CSV и TXT.

Для проверки объединения данных можно одновременно выбрать несколько CSV-файлов с одинаковой структурой столбцов.

При запуске приложения через Docker каталог `data/` подключается к контейнеру. Чтобы использовать собственный CSV-файл, необходимо поместить его в каталог `data/` проекта. Внутри контейнера файлы доступны по пути:

```text
/app/data
```

## Требования к CSV-файлам

Для совместной обработки нескольких CSV-файлов следует использовать файлы с одинаковой структурой столбцов.

Для выполнения статистического анализа необходимо выбрать:

1. поле группировки;
2. числовое поле, значения которого будут использоваться для расчёта статистических характеристик.

Для каждой сформированной группы программа рассчитывает:

- количество корректных числовых значений;
- сумму;
- среднее арифметическое;
- медиану;
- минимальное значение;
- максимальное значение.

Некорректные или отсутствующие числовые значения исключаются из статистической обработки.

## Установка и запуск

### Локальный запуск

Клонируйте репозиторий и перейдите в каталог проекта:

```bash
git clone <https://github.com/SaVOk111/CSV_Report_Generator>
cd CSV_Report_Generator
```

Создайте виртуальное окружение:

```bash
python -m venv .venv
```

Активируйте виртуальное окружение.

#### Windows

```powershell
.venv\Scripts\activate
```

#### Linux / macOS

```bash
source .venv/bin/activate
```

Установите зависимости:

```bash
pip install -r requirements.txt
```

Запустите приложение:

```bash
python src/main.py
```

После запуска откроется графическое окно приложения.

## Запуск через Docker

Для контейнеризированного запуска необходимо установить и запустить Docker Desktop или другую совместимую среду Docker.

Соберите образ и запустите контейнер из корневого каталога проекта:

```bash
docker compose up --build
```

После успешного запуска откройте в браузере:

```text
http://localhost:6080/vnc.html
```

Графический интерфейс приложения будет доступен через noVNC.

Для выбора демонстрационных или добавленных пользователем CSV-файлов откройте в файловом диалоге приложения каталог:

```text
/app/data
```

Чтобы использовать собственные CSV-файлы при запуске через Docker:

1. поместите необходимые файлы в каталог `data/` проекта;
2. откройте приложение через noVNC;
3. выберите файлы из каталога `/app/data`.

Каталог `data/` подключается к контейнеру, поэтому добавленные в него файлы доступны приложению.

Для остановки контейнера выполните:

```bash
docker compose down
```

## Запуск тестов
